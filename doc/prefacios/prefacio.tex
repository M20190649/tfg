\noindent{\textbf{Palabras clave}: ciudades inteligentes, sensores Bluetooth, sensores WiFi, floating car data, tráfico rodado, probe data
\vspace{0.7cm}

\noindent{\textbf{Resumen}\\

El gran avance en tecnologías móviles durante los últimos años ha propiciado que hoy en día la mayoría población utilice dispositivos móviles inalámbricos como \textit{smartphones} o \textit{smartwatchs}, estos dispositivos están emitiendo continuamente información al medio que permite identificarlos de manera única, surge la oportunidad de utilizar estos datos para analizar patrones de movimiento de la población que permitan optimizar las infraestructuras y los recursos de las ciudades con el fin de mejorar la calidad de vida de las personas y del entorno.\\

En este proyecto se ha desarrollado un sistema de detección de personas y vehículos utilizando un sensor de bajo coste  y reducido consumo energético basado en soluciones \textit{System on chip}, este sensor permite la detección de dispositivos inalámbricos en tiempo real mediante el análisis del tráfico WiFi y Bluetooth. Este sistema también cuenta con un servidor central al que los sensores envían las detecciones realizadas, desde donde se podrán visualizar a través de una página web o utilizando la API REST.\\

\noindent{\textbf{Abstract}\\

The advancement in mobile technologies in recent years, has led the majority of the population to use wireless mobile devices such as smartphones or smartwatchs, these devices are continuously broadcasting information that can be used to uniquely identify them, this bring us the opportunity to use these data to analyze population movement patterns that allow optimizing the infrastructure and resources of cities in order to improve the quality of life of people and the environment.\\

In this project, a system for detecting people and vehicles had been developed using a low cost and low energy consumption sensor based on System on chip solutions, this sensor allows detection of wireless devices in real time by analyzing WiFi and Bluetooth traffic. This system also has a central server to which the sensors send the detections made, from where they can be viewed through a web page or using the REST API.\\
